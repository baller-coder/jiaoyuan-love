# 03 · 调查研究

> **出处**：《反对本本主义》（1930 年 5 月，江西寻乌）
> **原文链接**：https://www.marxists.org/chinese/maozedong/marxist.org-chinese-mao-193005.htm

## 核心思想（一句话）

**没有调查，就没有发言权。情感问题里 3 个月的脑补 + 100 篇情感号文章，不如 30 秒侧面打听一次共同朋友。**

## 原文摘录

> "没有调查，没有发言权。"
>
> "调查就像'十月怀胎'，解决问题就像'一朝分娩'。调查就是解决问题。"
>
> "我们说马克思主义是对的，决不是因为马克思这个人是什么'先哲'，而是因为他的理论，在我们的实践中，在我们的斗争中，证明了是对的。"
>
> ——《反对本本主义》

## 适用场景

- 用户**凭脑补判断对方**，没有事实依据
- 用户从来**没问过共同朋友**侧面意见
- 用户**只看 PUA 教程 / 情感号文章**做决策，不看真实对象
- 用户**没有量化过**对方行为信号（主动次数、回复速度、肢体距离）
- 用户在做大决定（表白 / 分手 / 挽回）但**信息明显不足**

## 不适用场景

- 调查超出合理边界变成跟踪、偷窥、查通讯记录——这是越界
- 用户已经做了充分调查只是**回避做决定**——这时不是该多调查，是该上 `02-实践认识.md`
- 涉及对方隐私的调查（家庭成员病史、收入隐情等）需要尊重对方知情权

## 应用步骤

### 步骤 1：列出你当前的判断和依据

写下来：
- 我目前的判断是 X（"她对我有意思" / "她在玩我" / "他配不上我"）
- 我的依据是 Y（具体哪些事实 / 哪些"感觉"）

把"事实"和"脑补"分开。**99% 的情感焦虑里，"依据"那一栏其实是脑补。**

### 步骤 2：设计三种调查动作

**A. 行为信号量化**（你自己能做的）
- 数据化对方行为：3 个月里**她主动**发起几次（不是你发她回，是她发起话题/邀约）？
- 互动模式：回消息趋势变快还是变慢？告别时谁先转身？
- 边界信号：朋友圈给你看多少？带你见过她的朋友/家人/室友吗？

**B. 共同朋友侧面打听**
- 找一个共同朋友（最好是对方的同性密友），问一句"你觉得她对我什么感觉"
- 一句问话 30 秒，能省掉你 30 天的内耗
- **重要**：不要让朋友"传话"或"撮合"——只是收集信息

**C. 反向调查**
- 她对**其他男性朋友**也是这个距离感吗？也许她就是高边界感的人
- 她最近半年的状态？（忙工作 / emo 期 / 刚分手 / 搬家）
- **她是不是被你的脑补"演"成了一个并不真实的形象**？

### 步骤 3：反对本本主义

> "反对本本主义"原意是反对教条照搬书本。
> 在情感场景对应：**反对教条照搬 PUA 教程 / 情感号文章 / 你父母的恋爱观 / 你前任的相处模式**。

每一次决策前问一句：**我现在的判断，是基于眼前这个真实的人，还是基于哪本"书"里的模板？**

例子：
- 她回消息慢 → "情感号说回消息慢就是不喜欢" ❌（本本主义）
- 她回消息慢 → 她最近在准备项目答辩，所有人她都回得慢 ✅（具体调查）

### 步骤 4：调查的结论是新假设，不是终点

调查得到的结论不是"答案"，是"**更可信的假设**"。这个假设要拿去 `02-实践认识.md` 里用真实动作验证。

调查 → 假设 → 实践 → 修正假设 → 再实践。这才是完整循环。

## 常见误用

| 误用 | 为什么不对 |
|---|---|
| ❌ 调查变成跟踪 / 偷看手机 / 查通讯记录 | 越界且违法，且收集到的信息不能用 |
| ❌ 只问一个朋友就当定论 | 单一信源 = 单一偏见 |
| ❌ 用调查代替实践（光打听不行动） | 调查的结论是假设，需要实践验证 |
| ❌ 把调查到的事实再次脑补成结论 | 比如"她朋友说她最近忙" → 你脑补成"她忙着躲我" |
| ❌ 调查太多次反而暴露你的"在意" | 调查需要克制和分散，不要在同一个朋友那里反复打听 |
